# ABSTRACT

Large-scale Mixture-of-Experts (MoE) models have demonstrated remarkable capabilities across natural language processing and multimodal tasks, yet their practical deployment remains severely constrained by GPU memory bottlenecks. While MoE architectures promise efficient computation through sparse expert activation, their deployment requires simultaneous residency of all expert weight matrices in GPU VRAM—a requirement that incurs prohibitive memory costs (20–30 GB for state-of-the-art models) and renders such systems inaccessible on consumer-grade and edge hardware. Existing approaches either incur out-of-memory failures (DeepSpeed MoE on constrained hardware) or introduce severe latency penalties through synchronous disk-to-GPU staging (MoE-Infinity: 920 ms/token, ~5.4 GB pinned host memory allocation). This thesis presents CacheFlow, a comprehensive three-tier memory hierarchy system that decouples GPU memory requirements from total expert parameter count, enabling efficient MoE inference on resource-constrained hardware without modifying model structure or routing mechanisms. The core innovation chains disk-resident expert weights through CPU-pinned memory staging buffers to GPU VRAM slots via asynchronous, non-blocking PCIe transfers. The system incorporates direct disk I/O via safetensors slicing (MmapExpertStore), intelligent pinned memory buffer management (_next_staging()), and an LRU cache with intra-batch freezing (ExpertCache) to prevent inadvertent eviction of required experts during batch processing. Empirical evaluation demonstrates that CacheFlow achieves **5–9× reductions in peak GPU memory** compared to baseline systems across constrained capacity settings. Specifically, at capacity constraint C=1 (single expert slot per MoE layer), the system maintains a peak VRAM footprint of **988.17 MB** while achieving per-token latency of **598.23 ms**, improving to **462.27 ms/token at C=8** (1845.26 MB VRAM)—a **23% latency reduction**—with throughput scaling from **100.62 to 130.04 tokens/minute**. The intra-batch freezing mechanism adds negligible overhead (~0.1% latency) while guaranteeing computational correctness. By enabling deployment of state-of-the-art MoE models on consumer-grade GPUs (RTX 3060, Jetson Orin with 12 GB VRAM) previously requiring specialized infrastructure, CacheFlow democratizes access to sparse expert architectures and establishes a practical foundation for scalable, efficient MoE inference in resource-constrained environments.
